For local Andes RISC-V targets connected via AICE or Maverick, RVBuilder provides a graphical interface for programming application binaries into the target flash memory. To perform in-system programming using RVBuilder, proceed as follows:

1. In the **Explorer** view, right-click the desired project and select **RVBuilder: Flash Burner** on the drop-down menu. 

2. In the invoked **Flash Burner** settings UI, select the tab corresponding to your programming method and configure the required programming options.

3. Click the **Burn** button to start flash programming.
